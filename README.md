# collision2d
Provides basic *WIP* 2D collision detection for V.
See [this demo](https://github.com/scurty-labs/collision2d/blob/master/examples/demo.v) for example usage.

## Features
* point_on_line
* point_in_circle
* point_in_aabb
* line_in_circle
* line_in_aabb
* circle_meets_circle
* circle_meets_aabb
* aabb_meets_aabb

## Planned Features
* Simple Raycasting
* Rotated Rectangle Collision(S.A.T)
